Concept of WWW

  • Its stands for World Wide Web.
  • It is basically system of internet servers that support specially formatted documents.
  • It is collection of text pages, videos, digital, photographs and animations we can access over the internet
  • User access the World Wide Web facilities via client called a web browser

HTTP Protocol

  1. HTTP stands for Hypertext Transfer Protocol
  2. HTTP is based on the client-server architecture model and a stateless request/response protocol
  3. 404 File Not Found in a common HTTP status code.
